# Development

This document describes the process for running this application on your local computer.

## Getting started

You need [`pdftotext`](https://www.xpdfreader.com/pdftotext-man.html) to run the parsing script. `pdftotext` is included with [the `poppler` library](https://poppler.freedesktop.org/).

```shell
git clone https://github.com/stoyanK7/BG-DE-anki-decks.git
cd BG-DE-anki-decks
pipenv sync --dev
```
